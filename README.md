# EchoAI-Peds

This repository contains the official implementation of **EchoAI-Peds**, the first multi-task deep learning model for pediatric echocardiography analysis.

## Overview

**EchoAI-Peds** is a video-based vision transformer trained to simultaneously detect 28 congenital heart defects, structural and functional abnormalities, repairs, and interventions directly from complete pediatric echocardiography studies with multiple videos. 

## Preprint

Our work is available as a preprint on medRxiv:

🔗 https://www.medrxiv.org/content/10.1101/2025.10.27.25338912v1

## Model Weights

We are excited to release the model weights upon acceptance of our manuscript for final publication.

In the meantime, we provide example code (load_model.py) demonstrating how to load and use the model. The current scripts include placeholder paths to illustrate the expected setup.
